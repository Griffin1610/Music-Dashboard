# Music Dashboard

**Music Dashboard** is a C# application designed to organize and manage music tracks, podcasts, and other audio files. It offers an efficient way to categorize and handle audio content with a clean, modular structure.

## Features
- **Track Management**: Create and manage a database of music tracks.
- **Podcast Integration**: Handle podcasts with ease, including metadata and episode details.
- **Audio File Support**: Extend functionality to general audio files.
- **Rolling Stones Chart Integration**: Incorporates chart data for music categorization.

## Project Structure
- **`Track.cs`**: Defines the structure and behavior of music tracks.
- **`Podcast.cs`**: Handles podcast-specific logic and metadata.
- **`Audio.cs`**: A base class for managing audio-related data.
- **`Polly_Final.sln`**: The main solution file for Visual Studio.
